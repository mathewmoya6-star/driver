01
Open Market Value
Fair Market Value
Court disputes, pre-purchase negotiation, general market reference. Based on 5+ comparable transactions.
02
Forced Sale Value
Bank / Asset Finance
Secured lending, repossession scenarios. 20-30% below market value. Max LTV: 70-80% of FSV.
03
Replacement Value
Insurance Comprehensive
Agreed value for insurance policies. OMV + 10-15% incidental costs (dealer markup, transfer fees).
04
Pre-Accident Value
Total Loss Claims
Market value immediately before accident. Total loss threshold: repair cost > 75% of PAV.
05
Salvage Value
Post-Total Loss
Recoverable value from written-off vehicle. 15-30% of pre-accident value depending on parts demand.
06
Net Book Value
Asset Audit / Balance Sheet
Corporate accounting using 25% reducing balance method. Original cost × (0.75)^age.
07
Customs CRSP Value
KRA Import Duty
Statutory depreciation schedule. Age-based depreciation: 10% at 1 year, 60% at 8+ years.
08
Diminished Value
Post-Repair Claim
Reduced resale value after accident repairs. Stigma discount: 5-30% depending on damage severity.
09
Residual Value
Lease / Asset Finance
Projected value at end of lease term. 48% of original at 36 months for light vehicles.
10
Agreed Value
Custom / Contract
Mutually accepted value for classic, modified, or specialized vehicles. Supersedes market value.
